## Flags and Open Items

1. **Spelling — Monteray vs Monterey.** Stage 1 working drawings use **Monteray**; the Stage 2/3 folder structure uses **Monterey**. One of these is likely a typo. *Kyle to confirm which is correct so we can standardise across all KPV documentation.*

2. **Two Monterey variants (Kyle's observation).** Stage 2 contains two Monterey folders (122–124 and 133–135). Kyle has confirmed these are two distinct variants — **one single garage, one double garage**. Which folder maps to which variant is still [TBC] pending review of the drawings. The double-garage Monterey is not documented in Stage 1 — its bedroom count, bathroom count, and floor area all need to be captured from drawings.

3. **Berkeley unit-number overlap in Stage 2.** Folders `Berkeley 136-138` and `Berkeley 137-139` both reference units 137 and 138. Almost certainly a folder-naming error — *Kyle to confirm the correct unit ranges.*

4. **Berkeley / Monterey unit-number overlap in Stage 2.** Folders `Berkeley 122-124` and `Monterey 122-124` share the same unit numbers. Again almost certainly a naming error — *Kyle to confirm.*

5. **Specs inherited from Stage 1.** Where a Stage 2 or Stage 5 typology also appears in Stage 1, this document assumes the specs carry over unchanged. Once Stage 2 and Stage 5 consent drawings are available, every row should be re-verified — it is common for later-stage variants to have minor plan revisions.

6. **Stages 3 and 4 not included.** Per Kyle's scope, Stages 3 and 4 are omitted from this document. Stage 3 introduces three new typologies (Eastwood, Harris, Robertson) and Stage 4 introduces Stanaway — none of which have specs on file yet.

7. **Robertson specs.** Robertson first appears in Stage 3 folders, but is only included in this Stages 1/2/5 document because it also appears in Stage 5. No drawings are currently on file — specs are [TBC].
