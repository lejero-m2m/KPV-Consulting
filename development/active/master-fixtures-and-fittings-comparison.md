# Master Fixtures & Fittings — KPV Comparison

**Karaka Pines Villages — All Villages, All Typologies**
**Version:** v2 (KPV V8 Standard now extracted)
**Date:** 2026-05-11
**Purpose:** Compile a master Karaka Pines fixtures & fittings spec by comparing what each active builder currently delivers against the **KPV Standard Technical Specification V8** (the contractual baseline), so KPV can lock in deviations and produce an unambiguous master spec going forward.

---

## How to read this document

- **Item** — the spec category (e.g. "Main Cladding", "Shower Mixer")
- **KPV V8 Standard** — the canonical KPV spec from `Standard Technical Specification V8 2024 09.docx` (dated 11/09/2024). This is the contractually referenced baseline.
- **Mike Greer / Signature / Downey / G.J. Gardner** — what each builder's own template/contract specifies. **✅ ON-spec / ⚠️ DEVIATION / —** = not addressed by that builder.
- **KPV Master Spec** — *blank by design* — for Kyle to either confirm V8 as-is, or specify the agreed deviation per item

Cells marked **`—`** = item not addressed in that builder's spec. `[TBC]` = source not yet extracted in this version.
