# Overall Development Knowledge

Cross-cutting reference material that applies across all KPV village developments.

## Folders

| Folder | Content |
|--------|---------|
| `rma/` | Resource Management Act — legislation, guides, relevant case law |
| `building-standards/` | NZ Building Code, standards, compliance guides |
| `district-plans/` | District plan references (region-specific rules live in village folders) |
| `funding-and-finance/` | Funding models, financial frameworks, cost benchmarks |
| `consenting-guides/` | Consenting process guides, checklists, timelines |
| `market-research/` | Market analysis, demographic data, demand studies |

Add files to the relevant subfolder. These are shared resources — region-specific documents belong in their village folder.
